# Git & GitHub Workshop 🚀


Welcome!

Your task is to contribute to this repository by adding a markdown file about yourself.

## Steps

1. Fork this repository.
2. Clone your fork to your computer.
3. Create a new branch.
4. Create a markdown file inside the `students/` folder.
5. Name the file as:

```
your-name.md
```

Example:

```
abhishek-u-pai.md
```

6. Add the following content:

```md
# Your Name

**One good thing about me:**
I enjoy solving challenging problems.
```

7. Commit your changes.

```
git add .
git commit -m "Add my profile"
```

8. Push your branch.

```
git push origin <branch-name>
```

9. Open a Pull Request.

Congratulations! 🎉 You've made your first contribution.
